##  Live Application

This is the backend for the Phonebook application, deployed on Fly.io.

## 🛠 How to Run Locally

```bash
npm install
npm run dev

API Endpoints:
- GET /api/persons
- POST /api/persons
- DELETE /api/persons/:id
```
